# Ideen

- Hügelgrab (Kelten)
- Banditensafe mit sehr vielen Fallen, aber viel Loot
  - Fallgrube mit angespitzten Pfählen
  - Fallstrick mit Stein/angespitzten Pfählen
  - Unterwasserepisode (am Ende?)
  - Druckplatten
  - Automatische Schießscharten
- Pharaonengrab

  - Schmiedin
